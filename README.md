---

## 📖 Sobre o Projeto

O Read Quest não é apenas um simples rastreador de páginas. Ele foi construído com base na **Teoria do Loop do Hábito** (Deixa -> Rotina -> Recompensa) para ajudar os usuários a construírem ou evoluírem o hábito da leitura de forma orgânica e gamificada. 

O aplicativo oferece total liberdade criativa para o usuário customizar sua interface, diversas métricas para acompanhar o progresso e um sistema de recompensas único através de uma loja virtual integrada.

## ✨ Funcionalidades Principais

*   **🔄 Baseado no Loop do Hábito:** Sistema pensado para criar gatilhos saudáveis de leitura e recompensar o usuário ao fim de cada sessão.
*   **📊 Monitoramento Flexível:** Registre sua leitura da forma que preferir (por número de páginas, tempo de leitura ou porcentagem do livro).
*   **🎨 Interface 100% Customizável:** Adapte o app ao seu gosto. Mude cores, temas e a disposição das informações para tornar o ambiente de leitura "seu".
*   **🛒 Loja Virtual & Gamificação:** Ganhe pontos (moedas virtuais) ao bater metas e registrar leituras. Use esses pontos na loja do app para comprar itens virtuais, novos temas de interface, avatares ou selos de conquista.
*   **📈 Estatísticas Detalhadas:** Acompanhe seu ritmo de leitura, livros lidos no ano e sequências (streaks) de dias lidos.

## 🧠 Como aplicamos a Teoria do Loop do Hábito?

1.  **A Deixa (Gatilho):** Lembretes customizáveis e uma interface atrativa que convida o usuário a abrir o app.
2.  **A Rotina:** O ato de ler e utilizar as ferramentas de monitoramento amigáveis e sem fricção.
3.  **A Recompensa:** O sentimento de progresso visual (estatísticas) e o ganho de moedas para serem gastas na Loja Virtual.

## 🚀 Como executar o projeto localmente

### Pré-requisitos
Certifique-se de ter o seguinte instalado em sua máquina:
*   [Flutter SDK](https://docs.flutter.dev/get-started/install)
*   [Android Studio](https://developer.android.com/studio) ou [VS Code](https://code.visualstudio.com/) com as extensões do Flutter e Dart.
*   Um emulador Android/iOS configurado ou um dispositivo físico conectado.

### Passo a passo

1. Clone este repositório:
```bash
git clone https://github.com/AyronMT/Monitoramento-de-Leitura/
```

2. Acesse a pasta do projeto:
```bash
cd Projeto
```

3. Instale as dependências:
```bash
flutter pub get
```

4. Execute o aplicativo:
```bash
flutter run
```
